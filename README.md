# burn-evorl

> [!CAUTION]
> **Alpha Stage**: This project is in the early stages of development. Currently focusing on project structure and design.

Evolutionary Deep Reinforcement Learning library using [burn](https://github.com/tracel-ai/burn) and influenced by [burnrl-examples](https://github.com/yunjhongwu/burn-rl-examples). 

## Early Stages
The initial intent was to build the necessary components to replicate a chess playing agent in a manner similar to [AlphaZero: Shedding new light on chess, shogi, and Go](https://deepmind.google/blog/alphazero-shedding-new-light-on-chess-shogi-and-go/). This requires higher dimensional observation or state spaces and action spaces.

Simulataneously, while building out the first reinforcement learning prototype, I came accross the paper [Evolutionary Reinforcement Learning: A Systematic Review and Future Directions](https://arxiv.org/abs/2402.13296v1).
