# Appendix A — Evolutionary Computation Algorithms

This appendix gives derivations, pseudocode, and implementation notes for the
evolutionary computation algorithms in `rlevo-evolution`. It is the place to come
when the conceptual summaries in [Part I](../part-1-foundations/02-evolutionary-computation.md)
are not enough.

> This appendix is a stub. Algorithm pages will be added incrementally.

## Contents (planned)

- Genetic Algorithm — full pseudocode, operator derivations, convergence analysis
- Evolution Strategies — self-adaptive σ update, (μ, λ) vs (μ + λ) selection
- CMA-ES — covariance matrix update, path length control, step size adaptation
- UMDA / PBIL / cGA — marginal model update rules
- MIMIC — dependency chain construction, conditional entropy estimation
- BOA — Bayesian network structure learning (BIC scoring), ancestral sampling
- Gene Expression Programming — chromosome-to-tree decoding, recombination
- Memetic wrapper — local search operators, Lamarckian vs Baldwinian write-back
