# Part IV — Open Problems & Research Directions

`rlevo` is alpha software. It implements enough to be useful and enough to be
interesting as a research substrate — but it does not implement everything, and
some of what it does implement has known limitations.

This part is an honest accounting: where the library stands today, what the
active research questions are, and how to contribute if you want to push it
forward.
